About EZ Library

1. @ez-library/cypress: If you're familiar with Cypress, you'll love the convenience and simplicity offered by @ez-library/cypress. It provides a set of pre-built functions and utilities that abstract away the complexities of Cypress, allowing you to write clean and concise automation scripts. Whether you're testing web applications, APIs, or performing end-to-end tests, @ez-library/cypress simplifies the process and reduces the learning curve.

2. @ez-library/wdio:
If you're working with WebDriverIO (WDIO), @ez-library/wdio is the library you need. Designed to enhance the functionality and productivity of WDIO, @ez-library/wdio offers a comprehensive set of utilities and helper functions. It streamlines the test writing process, providing shortcuts for common actions, custom assertions, and seamless integration with popular testing frameworks. With @ez-library/wdio, you can achieve faster test automation and maximize the benefits of WebDriverIO.

3. @ez-library/playwright:
For those who prefer Playwright as their automation tool of choice, @ez-library/playwright is a game-changer. By leveraging Playwright's cross-browser capabilities, @ez-library/playwright enables you to write tests that can be executed on multiple browsers effortlessly. With its intuitive syntax and powerful features, you can automate complex scenarios with ease, making your testing process more robust and reliable.

4. Java Library, Selenium:
Enhanced Element Interaction: With eaSy, interacting with web elements becomes more intuitive. It provides simplified methods to perform common actions like clicking, typing, selecting options, and validating element properties. By encapsulating the complexities of locating and interacting with elements, eaSy makes your automation code more readable and maintainable. Advanced Wait Handling: Waiting for elements to appear or specific conditions to be met is crucial in automation testing. eaSy simplifies this process by offering explicit wait functions that handle timeouts and expected conditions seamlessly. You can easily wait for elements, page loads, text presence, and more, ensuring stable and reliable test execution. Reporting and Logging: Keeping track of test execution and generating meaningful reports is essential for test analysis. eaSy includes built-in reporting and logging capabilities, making it easy to generate HTML or PDF reports, capture screenshots, and log test execution details. This feature helps in identifying and debugging issues effectively.
